# FXmonitor

WebブラウザでUSD/JPYの参考レートを大きく表示できる、シンプルなモニターです。
ウィンドウをモニターの端に置いてこまめに確認したり、Webブラウザが使用できる端末をUSD/JPY常時表示モニターにしたりできます。

## 使い方

次のページをWebブラウザで開くだけで利用できます。

https://kaico-hub.github.io/FXmonitor/

アプリのインストールや会員登録、APIキーの設定は必要ありません。
画面を開いている間は自動で最新のレートを取得し、表示されるレートはおおむね40秒に1度更新されます。
通信状況やデータの配信状況によって、更新のタイミングが前後することがあります。

## 画面表示について

- 白い数値：正常にレートを取得しています。
- オレンジ色の数値：一時的に更新できず、最後に取得したレートを表示しています。
- `ERROR`：最初のレートを取得できませんでした。通信環境を確認して再読み込みしてください。
- 数値の下の日時：最後にデータを正常に取得した時刻です。

## ご注意

表示しているのはCoinbaseが配信するUSD基準の参考レートです。
FX会社で実際に取引できるBid／Ask価格とは異なります。
また、週末など市場が閉じている時間帯には、直近の参考レートが表示されることがあります。
