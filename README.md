# USD/JPY Monitor

使っていないiPhoneをドル円チャート専用モニターにするための、GitHub Pages用ミニサイトです。

## 使い方

1. GitHubで `usd-jpy-monitor` などの名前でリポジトリを作る
2. この中身をアップロードする
3. Settings → Pages → Deploy from a branch
4. Branchを `main` / root にする
5. 発行されたURLをiPhoneで開く
6. Safariの共有ボタン → ホーム画面に追加

## iPhone側おすすめ設定

- 設定 → 画面表示と明るさ → 自動ロック → なし
- 横置きスタンドに置く
- 充電しながら使う
- 画面輝度は低め
